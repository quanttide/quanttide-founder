# Changelog

本文件记录 quanttide-founder 仓库的所有重要变更。

## 格式

基于 [Keep a Changelog](https://keepachangelog.com/) 格式：
- `Added`：新增功能
- `Changed`：变更现有功能
- `Deprecated`：即将移除的功能
- `Removed`：已移除的功能
- `Fixed`：Bug 修复
- `Security`：安全相关变更
