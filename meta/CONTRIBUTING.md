# 贡献指南 - Meta

## 更新方式

本目录由 AI 自动维护，更新时会自动同步以下内容：

### metadata.md

当子模块有新的 Release 时，AI 会自动更新 `metadata.md`：
- 添加新 Release 信息
- 更新 Release 日期
- 记录更新日志

### AGENTS.md

当项目结构变化时，AI 会更新本文件。

## 手动更新

如需手动更新，请遵循以下规范：

### metadata.md 格式

```markdown
# 子模块 Release 汇总

## 子模块 Release 列表

| 子模块 | 仓库 | 最新 Release | Release 日期 |
|--------|------|-------------|-------------|

## 更新日志

### YYYY-MM-DD
- 具体更新内容
```
