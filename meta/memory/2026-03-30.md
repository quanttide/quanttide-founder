## 语义记忆

### Agent 体系设计

qtcloud-finance 的核心设计：

- **参与对象**：银行流水、Beancount 账本、调账记录
- **智能体**：记账员（自然语言→Beancount）、对账员（核对差异）
- **流程**：记账 → 对账 → 调账 → 再对账

### reconciler 迭代

v1 → v2 演进：

- Beancount 解析：正则 → beancount.loader
- 匹配算法：日期+金额精确 → 窗口+容差+打分
- LLM 调用：普通 → JSON 模式
- 结果模型：单一 → ReconciliationReport（含 auto/llm 匹配）
